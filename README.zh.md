> 专为 Solana DeFi 打造的轻量级智能代理工具包

## 🚀 核心特性
🎯 **专注 Solana** - 为 Solana DeFAI 操作优化  
🪶 **轻量级** - 仅核心必要功能，无冗余  
🔌 **插件架构** - 核心+插件模块化设计  
🛠️ **可扩展** - 轻松创建自定义插件/脚本  
🌐 **多语言** - 原生 Rust + TypeScript 支持  

### 为什么支持双语言?
| Rust | TypeScript |
|------|------------|
| ✅ 原生 Solana 程序交互 | ✅ Web3/DApp 集成 |
| ✅ 高性能底层 | ✅ 浏览器兼容 |
| ✅ 安全关键操作 | ✅ 快速原型开发 |
