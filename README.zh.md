## 🚀 核心特性
🎯 **专注 Solana** - 为 Solana financial protocols 优化  
🪶 **轻量级** - 仅核心必要功能，无冗余  
🔌 **插件架构** - 核心+插件模块化设计  
🛠️ **可扩展** - 轻松创建自定义插件/脚本  
🌐 **多语言** - 原生 Rust + TypeScript + Python 支持
